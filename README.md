## Optical Character Recognition App
Translate your Image into Text.

### Requirements 
- python: 3.10
- fastapi: 0.115.2
- uvicorn: 0.31.1
- python-multipart: 0.0.12
- aiofiles: 24.1.0
- pytesseract: .3.13 
- isort: 5.13.2

#### Install dependencies 
`pip install -r requirements.txt`

#### Clone the repository
`git clone https://github.com/vickyun/OCR`

#### Launch the WebApp
`fastapi run` or
`fastapi dev`
 for debugging mode
